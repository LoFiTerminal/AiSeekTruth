Hybrid Relay Mode: Truly Decentralized

Unlike traditional messaging apps that rely on central servers, AiSeekTruth transforms every installation into both a client and a relay. This creates a self-sustaining mesh network where users strengthen the system simply by participating.

‚úì Self-Sustaining
Network grows stronger with each user. No company infrastructure required.
‚úì Censorship Resistant
Impossible to shut down all user relays. Messages find alternate routes automatically.
‚úì Privacy Preserved
Relay apps store only encrypted envelopes. Messages remain end-to-end encrypted.
‚úì Local Discovery
Multicast finds nearby users on same WiFi. Chat without internet connection.

// Generate ephemeral keypair for ECDH
const ephemeralKeyPair = crypto_box_keypair();

// Derive shared secret using X25519
const sharedSecret = crypto_scalarmult(
  ephemeralKeyPair.secretKey,
  recipientPublicKey
);

// Encrypt message with XSalsa20-Poly1305
const nonce = crypto_box_random_nonce();
const ciphertext = crypto_box(
  messageBuffer,
  nonce,
  recipientPublicKey,
  senderSecretKey
);

// Sign with Ed25519 for authenticity
const signature = crypto_sign_detached(ciphertext, signingKey);

return { ciphertext, nonce, signature };
