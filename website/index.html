Core Features

root@security:~$

MILITARY-GRADE ENCRYPTION

Signal Protocol end-to-end encryption (Ed25519 + X25519 ECDH). Messages encrypted with XSalsa20-Poly1305. Keys derived with Argon2id. Even we can't read your messages.

crypto_sign_keypair() ‚Üí Ed25519
crypto_scalarmult() ‚Üí X25519 ECDH
root@network:~$

PEER-TO-PEER MESH

Decentralized Gun.js mesh network. No central servers to shut down. Messages route through multiple peers. Censorship-resistant by design. If one node falls, the network adapts.

gun.get('messages').on() ‚Üí P2P sync
distributed.hash.table ‚Üí mesh routing

Security Architecture

üîí

  • Signal Protocol

    Battle-tested encryption used by Signal, WhatsApp, and governments worldwide. Forward secrecy ensures past messages stay encrypted even if keys are compromised. Ed25519 signatures prevent message tampering.

  • No Metadata

    Traditional messengers leak who talks to whom, when, and for how long. AST uses P2P routing so no central server can build your social graph. Your relationships are private.

  • Open Source

    All code is public (GPL-3.0). Security researchers can audit every line. No backdoors, no telemetry, no hidden tracking. Trust through transparency. Fork it. Modify it. Own it.

  • Local-First

    Messages encrypted and stored on your device. No cloud backups. No AI reading your chats to "improve the product." Your device is the source of truth.
